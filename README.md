# Movie Maker

Cinema production web prototype.

## Usage

Open `index.html` in a browser.

Upload an image and enter a prompt. The page automatically chooses a motion style from the prompt, renders a short `.webm` video, and enables preview/download.

`Generate` is still available if you want to render again manually.

## GitHub Pages

This repository includes a GitHub Pages workflow at `.github/workflows/pages.yml`.

After pushing to `main`, GitHub Actions deploys the static site. The expected project URL is:

`https://gaja-trip.github.io/Movie-maker/`
